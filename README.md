# Tic-Tac-Toe-2D
Tic Tac Toe is a 2D game crafted in Unity 6, offering both 1 vs 1 Local Multiplayer and 1 vs PC Single Player modes. Enjoy a sleek UI with a 3x3 grid where players click to place their O or X symbols. Winning cells light up, accompanied by immersive background music and sound effects for clicks, wins, and draws. Simple, fun, and polished for a classic gaming experience!
